This is the API

findMembers

Returns all members

URL

https://staging.vestorly.com/api/v2/members

HTTP Method

GET

Response Type

Members

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

createMember

Create a new member in the Vestorly Platform

URL

https://staging.vestorly.com/api/v2/members

HTTP Method

POST

Response Type

Memberresponse

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Body: member Member

    Member you want to create

findMemberByID

Returns a single member

URL

https://staging.vestorly.com/api/v2/members/{id}

HTTP Method

GET

Response Type

Memberresponse

Parameters

  • Path: id String

    Mongo ID of member to fetch

  • Query: vestorlyAuth String

    Vestorly Auth Token

updateMemberByID

URL

https://staging.vestorly.com/api/v2/members/{id}

HTTP Method

PUT

Response Type

Memberresponse

Parameters

  • Path: id String

    Mongo ID of member to fetch

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Body: member Member

    Member you want to update
