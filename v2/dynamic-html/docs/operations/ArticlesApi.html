This is the API

findArticles

Returns all articles

URL

https://staging.vestorly.com/api/v2/articles

HTTP Method

GET

Response Type

Articles

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Query: limit Integer

    Limit on the number of articles to return

  • Query: textQuery String

    Search query parameter

  • Query: sortDirection String

    Direction of sort (used with sort_by parameter)

  • Query: sortBy String

    Field on model to sort by

findArticleByID

Returns a single article

URL

https://staging.vestorly.com/api/v2/articles/{id}

HTTP Method

GET

Response Type

Articleresponse

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Path: id String

    Article Id to fetch
