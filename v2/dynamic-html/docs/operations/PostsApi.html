This is the API

findPosts

Query all posts

URL

https://staging.vestorly.com/api/v2/posts

HTTP Method

GET

Response Type

Posts

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Query: textQuery String

    Filter post by parameters

  • Query: externalUrl String

    Filter by External URL

  • Query: isPublished String

    Filter by is_published boolean

createPost

Create a new post in the Vestorly Platform

URL

https://staging.vestorly.com/api/v2/posts

HTTP Method

POST

Response Type

Postresponse

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Body: post PostInput

    Post you want to create

getPostByID

Query all posts

URL

https://staging.vestorly.com/api/v2/posts/{id}

HTTP Method

GET

Response Type

Postresponse

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Path: id String

    ID of post to fetch

updatePostByID

Update A Post

URL

https://staging.vestorly.com/api/v2/posts/{id}

HTTP Method

PUT

Response Type

Postresponse

Parameters

  • Query: vestorlyAuth String

    Vestorly Auth Token

  • Path: id String

    id of post to update

  • Body: post Post

    Post you want to update
