This is the API

index

This lists all the active members for a given advisor

URL

http://developers.vestorly.com/api/advisors/{advisor_id}/members.json

HTTP Method

GET

Response Type

void

Parameters

    Authentication token

    Id of Advisor

    Fuzzy search by name

    Sort by: first_name, last_name, content_views, shares, posts_read, signed_up, last_active

    Sort order: asc, desc
