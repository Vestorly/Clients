This is the API

index

This retrieves a list of post objects

URL

http://api.vestorly.com/api/advisors/{advisor_id}/posts.json

HTTP Method

GET

Response Type

void

Parameters

    Authentication token

    Supports [all/published/unpublished/features/vestorly/newsletter]

    Filter by group id

    Filter by newsletter

URL

http://api.vestorly.com/api/advisors/{advisor_id}/posts.json

HTTP Method

POST

Response Type

void

Parameters

    Authentication token

    Id of advisor

    JSON block of post data

GET

Response Type

void
